# 附录B 快速参考卡片

## 学习目标
- [ ] 用一页内容快速回忆常见认知模型

## 关键概念
- 卡尔曼增益
- 精度加权预测误差

- 嵌套模型
- 层次模型

## 正文

### B1 经典模型一句话速记

| 模型 | 一句话理解 | 最常见数据 |
|-----|------|------|
| 线性回归/SEM | 看哪些变量能预测结果 | 连续因变量 |
| 效用模型 | 看价值与概率如何驱动选择 | 二选一/赌博选择 |
| 强化学习 | 看试错反馈如何更新价值 | 多试次选择 + 反馈 |
| DDM | 看证据积累如何决定反应时与准确率 | 选择 + RT |
| 卡尔曼滤波器 | 在噪声里持续追踪隐藏状态 | 连续观测序列 |
| 贝叶斯推理 | 用先验和证据更新信念 | 推理/判断任务 |
| HGF | 同时学习结果规律和环境波动性 | 反转学习/波动环境 |
| 吸引子模型 | 用动力系统解释稳定状态与切换 | 时间序列/神经行为联合 |

### B2 参数解释速查

| 参数 | 常见模型 | 心理学含义 |
|-----|------|------|
| $\alpha$ | RL | 学习率（新反馈权重） |
| $\beta$ | 效用/RL | 决策确定性（逆温度） |
| $v$ | DDM | 证据质量/积累速度 |
| $a$ | DDM | 谨慎程度（阈限） |
| $Q, R$ | Kalman | 环境波动 vs 观测噪声 |
| $K_t$ | Kalman | 动态学习率 |
| $\omega, \kappa$ | HGF | 波动性先验与层级耦合 |
| $w_+, w_-$ | 吸引子 | 自激励与竞争抑制强度 |

### B3 选模型 30 秒决策卡

1. 只要预测，不关心过程：线性回归/SEM。  
2. 关心价值选择，不看反应时：效用模型或RL。  
3. 同时关心准确率和RT：DDM。  
4. 关心噪声下的连续状态估计：卡尔曼滤波器。  
5. 关心“环境是否在变快”：HGF。  
6. 关心状态稳定、突变和竞争：吸引子模型。

### B4 常见误区排雷

1. AIC/BIC 更好不代表模型机制一定真实。  
2. 参数可拟合不代表参数可解释，要做参数恢复。  
3. 只报告最优模型不够，建议报告候选模型集合和比较标准。  
4. HGF 参数多，样本和试次数不足时容易不稳定。  
5. 吸引子模型通常需要更强的时间序列或神经约束。

## 小结
- 快速选模型先看研究问题类型，再看数据结构（是否有RT、是否是动态环境、是否需要层级不确定性）。
- 把参数解释写清楚，比只报告拟合指标更有理论价值。
