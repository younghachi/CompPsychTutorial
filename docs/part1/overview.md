# 第一部分 学术生存基础 - 概述

## 本部分的核心问题

当你进入一个新的科研领域时，如何快速建立扎实的理论基础？如何避免重复已有的工作？如何与AI和专业工具协作，提高学习与研究效率？

这第一部分将回答这三个问题。

## 学习目标

完成本部分的学习后，你应该能够：

- 独立进行文献搜索，找到高质量的学术论文
- 高效地阅读和管理学术文献，建立个人知识库
- 有效地利用AI作为学术助手，加速研究过程
- 理解学术工作的基本规范与逻辑

## 核心概念导读

### 学术文献的质量维度

在学术研究中，"好文献"有三个关键维度：

1. **期刊质量**：发表在Nature、Cell等顶级期刊的文献，其审稿标准与发表难度都更高
2. **作者影响力**：工作来自知名大学或研究机构、由著名学者主导的研究
3. **被引用量**：被后续研究广泛引用的文献说明其对学科的贡献

这三个维度构成了"高质量文献"的标准。

### 为什么需要系统的知识管理

科学研究本质上是"站在巨人的肩膀上"。有效的知识管理能帮助你：
- **减少重复工作**：之前的实验是否已有人做过？
- **找到创新空间**：现有研究中还有什么gap？
- **加速学习**：快速掌握新领域的基础知识

### AI的学术角色

AI不是替代你做研究，而是：
- **思想伙伴**：帮你检验、改进和扩展你的想法
- **学习助手**：用你能理解的方式讲解复杂概念
- **生产力工具**：加速编程、写作、数据处理等机械工作

理解这一点很重要，因为过度依赖AI会让你丧失对研究的控制权。

## 与后续部分的联系

学完第一部分后，你拥有了进入"实战研究"所需的基础知识与工具。

- **部分2-4**：将教你**如何规范地设计与开展研究**（实验设计、统计分析、计算模型、神经科学）
- **部分5-7**：将教你**如何用技术工具实现这些研究**（编程、工具使用、项目实战）

第一部分是后续部分的基石——如果你想做计算心理学研究，你首先需要知道这个领域已有哪些工作，它们是如何做的，这正是第一部分所教的内容。

## 常见问题

### Q1: 我可以跳过第1-2章直接学第3章吗？
**A**: 不建议。虽然第3章讨论AI，但它的实际应用依赖于你对学术规范的理解。而且，不了解文献的背景，即使有AI帮助你设计实验，你也难以判断这个设计是否创新。

### Q2: 学完这一部分就足够了吗？
**A**: 学完这一部分，你掌握了"做研究前的准备工作"。但要真正做出好研究，你还需要：
- 理解实验设计的规范（第2部分）
- 学会数据分析与建模（第3部分）
- 掌握具体的工具使用（第5-6部分）

### Q3: 我的研究领域不太一样，这些内容还适用吗？
**A**: 完全适用。无论你是做心理学、神经科学、医学还是其他领域的研究，文献搜索、文献管理、与AI协作的基本原理都是相通的。具体工具可能有差异，但方法论是一致的。
